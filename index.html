2 Steps to add a CSSDevice to your page

1) Download and include the CSS file into your page:

<link rel="stylesheet" type="text/css" href="devices.min.css">

2) Copy & Paste the following code where you want the device:

iPhone

<div class="cd-mobile cd-5s-silver">
  <div class="cd-cam"></div>
  <div class="cd-ear"></div>
  <div class="cd-home"></div>
  <div class="cd-screen">
    (your screen content goes here)
  </div>
</div>

iPad

<div class="cd-tablet cd-white">
  <div class="cd-cam"></div>
  <div class="cd-home"></div>
  <div class="cd-screen">
    (your screen content goes here)
  </div>
</div>

Macbook

<div class="cd-laptop">
  <div class="cd-top"></div>
  <div class="cd-bottom"></div>
  <div class="cd-cam"></div>
  <div class="cd-notch"></div>
  <div class="cd-screen">
    (your screen content goes here)
  </div>
</div>

Browser

<div class="cd-browser" style="width:600px;height:400px;"><!--adjustable browser size -->
  <div class="cd-nav">
    <span class="cd-red"></span>
    <span class="cd-yellow"></span>
    <span class="cd-green"></span>
    <div class="cd-url">
      http://Your-URL-Here.com
    </div>
  </div>
  <div class="cd-page">
    (your page content goes here)
  </div>
</div>

Helpers:

Fill Helper

When using an image, iframe, or anything else you want fullscreen on the device, give it the class "cd-fill" to have it fill the device:

<div class="cd-mobile cd-5s-silver">
  ...
  <div class="cd-screen">
    <img class="cd-fill" src="image.png" />
    <!-- <iframe class="cd-fill" src="http://example.com" frameborder="0"></iframe> -->
  </div>
</div>

Center Helper

To center the device in its parent element just give it the "cd-center" class:

<div class="cd-laptop cd-center">
  ...
</div>
