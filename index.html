3 Steps to add a CSSDevice to your page.

1: Download and include the CSS file into your page:

<link rel="stylesheet" type="text/css" href="devices.css">

2: Page the following code into where you want the device:

<div class="cd cd-5s-silver">
  <div class="cam"></div>
  <div class="ear"></div>
  <div class="home"></div>
  <div class="screen"></div>
</div>

3: Pick your device color and size and add the to the first divs' class:

Devices: cd-5s-silver cd-5s-gold cd-5s-black cd-5c-blue cd-5c-green cd-5c-red cd-5c-yellow cd-5c-white

Sizes: cd-lg (defaults to medium size) cd-sm cd-xs
