3 Steps to add a CSSDevice to your page

1) Download and include the CSS file into your page:

<link rel="stylesheet" type="text/css" href="devices.min.css">

2) Paste the following code into where you want the device:

<div class="cd cd-5s-silver [add optional size class here]">
  <div class="cd-cam"></div>
  <div class="cd-ear"></div>
  <div class="cd-home"></div>
  <div class="cd-screen">
    (your screen content goes here)
  </div>
</div>

3) Pick your device color & size. Then add them to the first divs' class:

Devices:

Helpers:

Image Helper

When using an image in the screen, give it the class "cd-image" to have it fit the screen:

<div class="cd cd-5s-silver">
  <div class="cd-cam"></div>
  <div class="cd-ear"></div>
  <div class="cd-home"></div>
  <div class="cd-screen">
    <img class="cd-image" src="image.png" />
  </div>
</div>

Center Helper

To center the device in its parent element just give it the "cd-center" class:

<div class="cd cd-5s-silver cd-center">
  <div class="cd-cam"></div>
  <div class="cd-ear"></div>
  <div class="cd-home"></div>
  <div class="cd-screen"></div>
</div>
