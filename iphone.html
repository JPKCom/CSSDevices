2 Steps to add iPhone to your page

1) Download and include the CSS file into your page:

<link rel="stylesheet" type="text/css" href="devices.min.css">

2) Copy & Paste the following code where you want the device:

iPhone

<section class="cd-mobile cd-5s-silver">
  <div class="cd-cam"></div>
  <div class="cd-ear"></div>
  <div class="cd-home"></div>
  <div class="cd-screen">
    (your screen content goes here)
  </div>
</section>

Option Classes:

All option classes should be added to the first divs class attribute.

Device sizes:

cd-lg
(medium size is default)
cd-sm
cd-xs

iPhone model/color options:

cd-5s-silver
cd-5s-gold
cd-5s-black
cd-5c-blue
cd-5c-green
cd-5c-red
cd-5c-yellow
