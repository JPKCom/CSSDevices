2 Steps to add Apple Watch to your page

1) Download and include the CSS file into your page:

<link rel="stylesheet" type="text/css" href="devices.min.css">

2) Copy & Paste the following code where you want the device:

iPad

<div class="cd-watch cd-center cd-rosegold cd-navy-band">
  <div class="cd-bracket cd-top"></div>
  <div class="cd-band cd-top"></div>
  <div class="cd-bracket cd-bottom"></div>
  <div class="cd-band cd-bottom"></div>
  <div class="cd-body"></div>
  <div class="cd-crown"></div>
  <div class="cd-button"></div>
  <div class="cd-screen">
    (your screen content goes here)
    <img class="cd-fill" src="img/watch.png">
  </div>
</div>

Option Classes:

All option classes should be added to the first divs class attribute.

Device sizes:

cd-lg
(medium size is default)
cd-sm
cd-xs

Watch color options:

cd-gold(default)
cd-silver
cd-rosegold

Watch band color options:

cd-red-band(default)
cd-black-band
cd-navy-band
